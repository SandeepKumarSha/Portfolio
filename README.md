## ✨ About

A modern, responsive **personal portfolio website** built with vanilla HTML, CSS, and JavaScript — featuring dark/light theme toggling, particle background animations, scroll-reveal effects, and a typing hero tagline. Designed to showcase my journey as an **AI/ML Engineer & Data Science Enthusiast**.

---

## 🚀 Features

- **🎨 Dark / Light Theme** — Toggle between themes with smooth transitions; preference is saved in `localStorage`.
- **🌌 Particle Background** — Interactive canvas-based particle animation that responds to window resizing.
- **⌨️ Typing Animation** — Auto-cycling typed text effect on the hero section (AI/ML Engineer, Data Scientist, etc.).
- **📜 Scroll Reveal** — Section elements animate into view as you scroll using IntersectionObserver.
- **📱 Fully Responsive** — Mobile-first design with a hamburger menu for smaller screens.
- **📬 Contact Form** — Functional contact form with client-side validation and status feedback.
- **🔗 Smooth Scrolling** — Navbar links smoothly scroll to the respective sections.
- **🏷️ SEO Optimized** — Proper meta tags, Open Graph tags, semantic HTML5, and heading hierarchy.

## ⚡ Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/SandeepKumarSha/Portfolio.git
   cd Portfolio
   ```

2. **Open in browser**
   ```bash
   # Simply open the HTML file — no build tools required!
   start index.html        # Windows
   open index.html          # macOS
   xdg-open index.html     # Linux
   ```

3. **Or use a local server** (recommended for best experience)
   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx serve .
   ```
   Then visit `http://localhost:8000` in your browser.
